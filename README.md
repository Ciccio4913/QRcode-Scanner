# 📱 QR Code Scanner

A simple Android application that scans QR codes using the device camera and displays the result instantly.

## 🚀 Features
- Scan QR codes using the camera
- Display scanned content in real time
- Automatically open links in the browser
- Clean and modern user interface

## 📦 Installation

1. Download the APK file from this path: app/release/QRcode Scanner.apk

2. Transfer the APK to your Android device

3. On your phone:
- Enable **"Install unknown apps"**
- Open the APK file

4. Install and launch the app

## 🛠️ Built With
- Java
- Android Studio
- ZXing Library

## 📌 Notes
- The app requires camera permission to scan QR codes
- Works on most Android devices
